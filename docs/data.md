## Synthetic Claims (`data/synthetic/claims.csv`)

Generated by `scripts/data/synthetic_generator.py` at startup if missing.
Seed: `random.seed(42)` — fully reproducible.

**250 claims, 80 synthetic members (`SYN-MEMBER-XXXX` format).**

Drug universe: top 40 brand drugs by NADAC unit cost delta (tablets/capsules preferred), drawn from `marts.switch_candidates`.

| Field | Description |
|-------|-------------|
| claim_id | `CLM-XXXXXX` |
| member_id | `SYN-MEMBER-XXXX` |
| age_band | 18-34, 35-44, 45-54, 55-64, 65-74, 75+ |
| sex | M or F |
| zip3 | 3-digit zip prefix (synthetic) |
| plan_id | PLAN-GOLD-001, PLAN-SILVER-002, PLAN-BRONZE-003 |
| drug_name | Brand drug name from NADAC |
| brand_generic_flag | Always "B" in synthetic claims |
| ndc | Brand NDC from NADAC |
| quantity | 30, 60, or 90 units |
| days_supply | = quantity (1 unit/day assumption) |
| fill_date | Random date in 2024 |
| paid_amount | NADAC brand cost × quantity × jitter (0.85–1.15) |
| member_cost_share | 5–30% of paid_amount |
| pharmacy_id | PHX-XXXX |
| prescriber_id | NPI-XXXXXXX |
| diagnosis_group | CARDIOVASCULAR, DIABETES, MENTAL_HEALTH, RESPIRATORY, MUSCULOSKELETAL, GASTROINTESTINAL, ONCOLOGY, OTHER |
| adherence_score | Random 0.40–1.00 per member |
| prior_switch_failure_flag | 1 with 15% probability |
| estimated_event_cost | $200–$2500 (ONCOLOGY/MENTAL_HEALTH: $1500–$8000) |
| preferred_pharmacy_available | 1 with 75% probability per member |
| pharmacy_access_score | Random 0.30–1.00 per member |

## NADAC Pricing (`NADAC.csv`)

Source: CMS NADAC (National Average Drug Acquisition Cost), as of April 29, 2026.
30,292 rows. Updated by CMS weekly/monthly.

Key columns used:
- `NADAC Per Unit` → unit cost ($/EA, $/ML, $/GM)
- `Classification for Rate Setting` → B (brand), G (generic), B-BIO, B-ANDA
- `Corresponding Generic Drug NADAC Per Unit` → fallback generic cost signal
- `Pricing Unit` → EA (tablet/capsule), ML (liquid), GM (topical)

## FDA Orange Book (`Orange_book_data_files/`)

Source: FDA Approved Drug Products with Therapeutic Equivalence Evaluations.
Files are tilde-delimited text, updated monthly by FDA.

- `products.txt` (48,083 rows): drug products, TE codes, brand vs. generic classification
- `patent.txt` (20,858 rows): patent expiry data
- `exclusivity.txt` (2,054 rows): market exclusivity codes

See `docs/data_dictionary/orange_book_dictionary.md` for full field definitions.

## DuckDB Warehouse (`data/warehouse/pharmaflow.duckdb`)

Built at startup by `scripts/services/data_service.ensure_warehouse()`.

| Schema | Table | Description |
|--------|-------|-------------|
| `raw_fda` | `orange_book_products` | Raw Orange Book products |
| `raw_fda` | `orange_book_patents` | Raw patent data |
| `raw_fda` | `orange_book_exclusivity` | Raw exclusivity data |
| `raw_nadac` | `nadac_prices` | Raw NADAC pricing |
| `marts` | `orange_book_products_clean` | Cleaned, typed OB products |
| `marts` | `therapeutic_equivalence_groups` | Groups by ingredient+form+strength+TE |
| `marts` | `switch_candidates` | Brand→generic pairs with NADAC costs |
